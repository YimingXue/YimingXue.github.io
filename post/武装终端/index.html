在ubuntu中使用zsh与oh-my-zsh

安装和配置

  • 查看ubuntu中默认安装了哪些shell
cat /etc/shells
  • 查看当前正在运行的是哪个版本的shell
echo $SHELL
  • 安装zsh、git和wget
sudo apt-get install zsh git wget
  • 安装oh-my-zsh
wget --no-check-certificate https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | sh
  • 替换bash为zsh并重启生效
chsh -s /bin/zsh
sudo reboot

使用手册

oh-my-zsh的github项目在此：oh-my-zsh，vim ~/.zshrc可以查看oh-my-zsh的配置文件，主要可以修改两个参数：

  • ZSH_THEME='ys'，这个修改oh-my-zsh的主题
  • plugins=(git last-working-dir extract z)，这个为oh-my-zsh添加一些插件

常用操作

  • 转换路径时可以省去cd命令，直接输入路径即可
  • 输入d可以查看访问过的路径，然后输入数字就可以直接切换
  • ctrl+L = clear = 清屏
  • zsh_stats，查看使用频率前20的命令
  • ctrl+u 清除一行的命令

添加插件

  • zsh-autosuggestions

    git clone git://github.com/zsh-users/zsh-autosuggestions $ZSH_CUSTOM/plugins/zsh-autosuggestions
plugins=(zsh-autosuggestions)
vim ~/.zshrc # 添加插件名字
source $ZSH_CUSTOM/plugins/zsh-autosuggestions/zsh-autosuggestions.zsh
source ~/.zshrc

终端分屏利器Tmux

安装和配置

sudo apt-get install tmux
# 启动Tmux
tmux
# 关闭Tmux
exit

重要概念

tmux有三个基本概念：会话(Session)，窗口(Window)和面板(Pane)。当你输入tmux后， tmux实际做的事是首先创建一个会话(Session)，然后在这个会话中创建一个窗口，你可以继续创建多个窗口(Window)，每个窗口初始只包含一个面板，继续分屏后，会出现多个面板(Pane)，你在其中看到的终端实际上都属于tmux的某个面板。更进一步讲，Session可以包含多个Window，每个Window又可以包含多个Pane。

快捷键备忘录

  • 列出所有快捷键： c-b ?
  • detach当前会话： c-b d （可通过tmux attach重新进入之前会话）
  • 选择并切换会话： c-b s

Session操作

  • 创建一个新的session： tmux new-s <name-of-my-session>
  • 在当前session中创建一个新的session，并保证之前的session依然存在： c-b 之后输入new-s <name-of-my-new-session>
  • 进入名为test的session： tmux attach -t test

Window操作

  • 创建一个新的窗口： c-b c
  • 关闭当前窗口： c-b &
  • 列出所有的窗口选择： c-b w
  • 切换到上一个窗口： c-b p
  • 切换到下一个窗口： c-b n
  • 使用窗口号切换窗口： c-b 窗口号（例如：1）
  • 重命名当前窗口： c-b ,

Pane操作

  • 横向分Terminal： c-b %
  • 纵向分Terminal： c-b “
  • 自由选择面板： c-b 方向键
  • 关闭当前pane： c-b x
  • 显示面板编号： c-b q
  • 列出所有会话： c-b s
  • detach当前session： c-b d

基操

  • 建立一个会话并且为其命名

    tmux new-session -s miforest

  • 建立一个会话并且同时指定这个会话中第一个窗口的名字

    tmux new-session -s miforest -n first

  • 重新attach一个session

    tmux attach -t work

  • 列出当前所有会话，在tmux中可以直接通过 c-b s 列出会话

    tmux ls

  • 创建会话时可以加上-d选项表示不自动attach到这个新会话

    tmux new-session -s miforest -d

  • 关闭会话

    tmux kill-session -t miforest

  • 直接创建窗口

    tmux new-window -n first -t miforest

  • 在创建会话和窗口时指定要执行的命令

    tmux new-session -s miforest -n first -d "ssh user@example.com"

  • 指定会话[:窗口:面板]发送任意按钮

    tmux send-keys -t miforest:editor 'emacs' C-m

Ubuntu Tmux 启用鼠标滚轮

  • ctrl+b再按:，进入命令行模式，然后输入set -g mouse on，这种状态下复制黏贴文本都需要==按住shift==再进行操作

常用插件使用

  • Undotree

    • 查看undo-tree

    • 启动方式 <leader>u

  • NERDTree文件导航工具

    • 基本操作
      key description
      ctrl+e 打开/关闭文件浏览器
      o 小写字母o，打开文件或者展开目录
      shift+c 即大写字母C，当前选中目录作为根目录
      u 上一层目录作为根目录
      :help NERDTree NERDTree帮助手册

  • NERDCommenter注释工具

    • 基本操作
      key description
      ,cc 注释一行
      ,cu 取消注释一行
      ,cA 在行末追加注释，注释模式取决于ca

  • neocomplcache自动补全工具

    • 基本操作
      key description
      ctrl+p 或 ctrl+n 在原始输入与补全方案间切换

  • Tagbar工具

    • 使用前要先安装exuberant-ctags，执行命令sudo apt-get install exuberant-ctags

    • 基本操作

      • 在你的项目根目录下运行，ctags -R *
      key description
      ctrl+] 转到定义处
      ctrl+t 返回
      ,tt 打开关闭tagbar，显示当前文件函数列表，变量列表、属性等
