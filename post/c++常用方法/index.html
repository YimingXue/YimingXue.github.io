vector使用总结

  • 使用时包含头文件： #include<vector>

  • C++中的STL指代的是Standard Template Library，标准模板库，惠普实验室开发的一系列软件的统称。

    vector的初始化

    #include<vector>
  
// 定义10个整型元素的向量，这种方法没有给出初值，此时库会创建一个值初始化的元素初值，
// 并把它赋给容器中的所有元素
vector<int> a(10);
  
// 定义10个整型元素的向量，并且给出每个元素的初值为1
vector<int> a(10, 1);
  
// 用b向量来创建a向量，整体复制性赋值
vector<int> a(b);
  
// 定义了a向量中的值为b中第0个到第2个元素
vector<int> a(b.begin(), b.begin()+3);
  
// 从数组中获得初值
int b[7] = {1,2,3,4,5,6,7};
vector<int> a(b, b+7);

    C++的vector元素去重

    第一种思路：

    • 首先对vector中的元素使用sort函数排序，使重复的元素排在相邻的位置
    • 接下来使用unique函数将重复的相邻元素放在vector末尾，返回值为末尾的第一个重复元素地址
    • 最后调用erase函数，删除重复元素
    #include<vector>
#include<algorithm>
using namespace std;
  
vector<int> v;
sort(v.begin(), v.end());
v.erase(unique(v.begin(), v.end()), v.end());

    第二种思路：

    • 直接使用set即可

        #include<set>
  #include<vector>
  using namespace std;

  vector<int> v{1,1,2,2,3,3};
  set<int> s(v.begin(), v.end());

map使用总结

  • #include<map>映射，键值对容器, unordered_map，不需要排序，更省时间

  • 所有元素会根据key值排序（默认升序，如果键值为字符串就是字典序），map中所有元素都是pair，同时拥有实值和键值，pair的first为键值，second为实值，底层将它的first作为红黑树的排序key

    // 创建map对象
map<string, double> m;
  
// 插入和访问（访问时如果map中不存在该键值，实值为空，对于数字则为0，字符串则为空串）
m["Jack"]  = 98.5;
cout << m["Jack"] << endl;
  
// 遍历
map<string, double>::iterator it;
for (it = m.begin(); it != m.end(); it++)
    cout << (*it).first << ":" << (*it).second << endl;
  
// 删除
m.erase("Jack");
  
// 清空
m.clear();
  
// 查找
map<string, double>::iterator it;
it = m.find("Jack");
cout << (*it).first << ":" << (*it).second << endl;
  
// 用count函数来判定键值是否出现，count函数的返回值只有两个，键值出现返回1，否则返回0
if (m.count("Jack"))
    cout << "FOUND" << endl;
else
    cout << "NOT FOUND" << endl;
  
// 前一种方法很直观，但是效率差很多，因为前面的方法需要执行两次查找，因此推荐后一种方法
if (m.count(key)>0){
    return m[key];
}
return null;
  
iter = m.find(key); // 返回迭代器，判断是否存在，不存在指向尾指针
if (iter != m.end()){
    return iter->second;
}
return null;

    vector<int>::iterator iter = find(v.begin(), v.end(), key);

unique函数

  • unique是C++标准模板库STL中十分实用的函数之一，使用此函数需要#include <algorithm>

  • 该函数的作用是“去除”容器或者数组中相邻的重复元素，但需要注意的是：

    • 这里的去除并非真正意义的erase，而是将重复的元素放到容器的末尾，返回值是去重后的尾地址
    • unique针对的是相邻元素，所以对顺序错乱的数组成员或者容器成员，需要先进行排序，可以调用std::sort函数
    #include <vector>
#include <algorithm>
  
int main(void)
{
    vector<int> v{1,2,5,3,4,5};
    std::sort(v.begin(), v.end());
    int len = std::unique(v.begin(), v.end()) - v; # 去重后的vector长度
}

lower_bound()和upper_bound()函数

lower_bound()和upper_bound()都是利用二分查找的方法在一个排好序的数组中进行查找的。

  • lower_bound(begin, end, val)：从数组的begin位置到end-1位置二分查找第一个大于或等于val的数字，返回该数字的地址，不存在则返回end。通过返回的地址减去起始地址begin，得到数字在数组中的下标。
  • upper_bound( begin,end,num)：从数组的begin位置到end-1位置二分查找第一个大于num的数字，找到返回该数字的地址，不存在则返回end。通过返回的地址减去起始地址begin,得到找到数字在数组中的下标。
