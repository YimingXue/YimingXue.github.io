使用文档

下载deb包
https://code.visualstudio.com/Download

安装
sudo dpkg -i code_1.19.2-1515599945_amd64.deb

运行
命令行执行code

修改语言

进入命令行：ctrl+shift+p

输入：Configure Language

修改："locale":"zh-CN"

概念

  • 配置文件
    • 单次运行一个脚本视为一个task，相应的配置文件为task.json
    • 整个文件夹或者多个文件夹视为一个工作空间，配置文件为settings.json
    • 调试环境的配置文件为launch.json
    • 这些配置文件是需要手动编辑的，编辑完保存好就会代替默认的配置文件生效

小技巧

  1. 查看函数或类的定义

    Ctrl+鼠标左键点击函数名或者类名即可跳转到定义处

    在函数名或者类名上按F2

  2. 更改变量名

    在变量名上按F2即可实现重命名变量

  3. python断点调试

    在行号的左边点击即可设置断点，在左边的调试界面可以查看变量的变化

不成功的方法(原因未知)

安装pip3

  • 安装pip3

    sudo apt-get install python3-pip

  • 升级pip3

    pip3 install --upgrade pip

  • 卸载pip3

    sudo apt-get remove python3-pip

安装python3.6

  • ubuntu14.04升级python3.4到3.6

    sudo add-apt-repository ppa:jonathonf/python-3.6

成功的方法，源码安装

这方法有效，解决了问题，唯一不太好的一点就是命令行要使用python3.6和pip3.6

# Remove existing python 3.6 if installed with apt
$ sudo apt-get autoremove python3.6

# Get the source
$ wget https://www.python.org/ftp/python/3.6.1/Python-3.6.1.tar.xz
$ tar xvf Python-3.6.1.tar.xz
$ cd Python-3.6.1

# Configure and install
$ sudo ./configure
$ sudo make altinstall

# Success!
$ pip3.6 -V
pip 9.0.1 from /usr/local/lib/python3.6/site-packages (python 3.6)

VSCode远程服务器编程

配置步骤

  • 在远程linux机器上安装rmate

    wget https://raw.githubusercontent.com/sclukey/rmate-python/master/bin/rmate
chmod +x ./rmate
mv ./rmate /usr/local/bin/rmate

  • 在本地windows上安装openssh

    • 直接利用window版的安装包：https://www.mls-software.com/opensshd.html

  • 在vscode中安装拓展remote vscode

    • 安装完成后，ctrl+shift+p，面板中点击Start Server，启动服务（这一步是必须的）
    • ssh -R 52698:127.0.0.1:52698 VIRTUAL_MACHINE_IP_ADDRESS
      • VIRTUAL_MACHINE_IP_ADDRESS是远程服务器的IP，52698是vscode默认的与服务器的连接端口，这个默认可以在setting里面查到
      • 例如：ssh -R 52698:127.0.0.1:52698 xueyiming@192.168.1.135 -p 22
    • 修改文件rmate -p 52698 linux中的文件名称
    • 设置自动启动，在setting.json文件中添加"remote.onstartup:true"
