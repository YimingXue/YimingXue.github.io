Win10在Jupyter Notebook中使用Tensorflow

安装好Tensorflow

  • 在激活Tensorflow后安装ipython和jupyter：conda install ipython，conda install jupyter
  • 安装python kernel for Tensorflow：ipython kernelspec install-self --user
  • 看到类似结果Installed kernelspec python3 in C:\Users\XXX\Jupyter\kernels\python3

参考资料：Windows环境下安装TensorFlow并在Jupyter notebook上使用

Windows下anaconda2与anaconda3共存

首先安装anaconda2，安装时两个选项打钩，之后将anaconda3安装在anaconda2的envs子目录下，这过程中取消那两个选项的勾选。安装完成后，在命令行输入python可以启动python2，使用activate python3后，此时命令都在python3下的环境，deactivate可以取消激活python3

Windows下anaconda创建环境与删除环境

  • Anaconda创建tensorflow环境
    • 打开Anaconda Prompt，然后输入conda create -n tensorflow python=3.5
    • 可以在Anaconda Navigator中的Environment中查看所有已经创建好的环境
  • 删除环境
    • conda remove -n tensorflow --all
