用户及文件权限管理

创建用户
sudo adduser lilei # 创建用户
su -l lilei # 切换用户
sudo passwd shiyanlou # 设置shiyanlou用户密码
用户组
# 查看自己属于哪些用户组
groups shiyanlou # 使用groups命令
cat /etc/group | grep -E "shiyanlou"

# 为用户添加用户组，主要是用来添加sudo权限
sudo usermod -G sudo lilei

# 删除用户
sudo deluser lilei --remove-home

Linux文件权限

查看文件权限
ls -l # 使用较长格式列出文件，ll是ls -l的别名
ls -lh # 更直观地查看文件的大小
ls -a # 查看隐藏文件
  • 文件类型和权限 链接数 所有者 所属用户组 文件大小 最后修改时间 文件名
    • 文件类型和权限：拥有者权限 所属用户组权限 其他用户权限
      • 变更文件所有者sudo chown 文件名 用户名
      • 修改文件权限chmod 777 文件名

Linux目录结构及文件基本操作

  • FHS（Filesystem Hierarchy Standard）标准，文件系统层次结构标准
    • Linux是以树形目录结构的形式来构建整个系统的，可以理解为树形目录是一个用户可操作系统的骨架。
    • FHS定义了系统中每个区域的用途、所需要的最小构成的文件和目录，同时还给出了例外处理与矛盾处理。
Linux文件的基本操作
touch 文件名 # 创建文件
mkdir 目录名 # 创建目录

tail -n 1 /etc/passwd # 查看passwd文件最后一行

输入 功能
删除字符
wd 从当前光标删除至下一个单词
d$ 从光标处删除至当前行尾部
dd 删除整一个当前行
p 输入p将最后一次删除的内容置入光标之后
0 移动光标到行首
r 输入r和一个字符替换光标所在位置的字符
cw 替换当前光标到单词的末尾的内容
c$ 替换当前光标到行末的内容
ctrl+g 实现当前编辑文件中当前光标所在位置以及文件状态信息
G 使当前光标直接跳转到文件最后一行
gg 使当前光标直接跳转到文件第一行
行号+G 光标跳转到行号位置
